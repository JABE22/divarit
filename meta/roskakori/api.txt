    TIETOKANNAN RAJAPINTA // OMAT FUNKTIOT

    Nämä funktiot on (tai pitäisi olla) lisättynä skeemalle.
    
    Lisäys tapahtuu ajamalla CREATE FUNCTION ... -tiedosto (hae erikseen!) esim. HeidiSQL:ssä.

    Kun funktio on lisätty, sitä voidaan kutsua kuin muitakin funktioita.
    
    > SELECT * FROM funktio_nimi(parameteri);








=== Funktiot [tietokanta nimi, parametrit ja palautettava arvo] ===


###  -- Tilauksen X painot eri divareittain ###

    tilauksen_painot(tid int)
    
        tid : tilauksen ID
    
    RETURNS TABLE(
        tilaus_id int, 
        divari_nimi varchar(20), 
        paino bigint,
        kappale_lkm bigint
    )
    
    
    
### -- Tilauksen X kaikki teokset ###

        tilauksen_teokset(tid int)
        
            tid : tilauksen ID
    
        RETURNS TABLE(
            tilaus_id int,
            tila int,
            kayttaja_email varchar(255),
            pvm date,
            divari_nimi varchar(20),
            kappale_id int,
            teos_isbn varchar(20)
        )
    
    
    
    
###  -- Hae käyttäjän X tiedot

    hae_kayttaja(hae_kayttaja(param_user_email varchar(255)))
    
        param_user_email : käyttäjän sposti (avain)
    
    
    RETURNS TABLE(
        email varchar(255),
        etunimi varchar(20),
        sukunimi varchar(60),
        osoite varchar(100),
        puhelin varchar(100),
        div_yllapitaja boolean
    )
    
    
### -- Tilauksen ID:n hakeminen

    hae_tilaus_id(param_kayttaja_email varchar(255)) 
        
        param_user_email : käyttäjän sposti (avain)
    

    Kuvaus: Hakee käyttäjän aktiivisen tilauksen (1. joka osuu vastaan).
    Jos tilausta EI ole, luodaa uusi. Palauttaa aina tilauksen ID:n
    
    RETURNS integer
    
    Esimerkkikutsu:
    > SELECT * FROM hae_tilaus_id('<sposti>') as tilaus_id;
    
    
### -- Teoksen lisäys ostoskoriin

    lisaa_ostoskoriin(p_kappale_id int, p_divari_nimi varchar(10), p_tilaus_id int) 
    
        p_kappale_id : kappaleen ID
        p_divari_nimi : divarin nimi 
        p_tilaus_id : tilauksen ID
        (kaikki pakollisia)
        
    Kuvaus: Tekee lisäyksen ostoskoriin, jos mahdollista
    
    RETURNS boolean
    
    Esimerkkikutsu:
    > SELECT * FROM lisaa_ostoskoriin(1234, 'D99', 1) as onnistuiko;


### -- Raportti 2. Myytävien kappaleiden hintatiedot

    hae_myytavien_hintatiedot()

    Kuvaus: Hakee MYYTÄVÄNÄ OLEVIEN (tila=0) teosten luokan, yhteishinnan (SUM), ja keskihinnan (AVG).
    Jos luokka on NULL, asetetaan arvoksi 'Luokittelematon'. Tulos järjestetään luokan nimen mukaan nousevaan järj.
    
    RETURNS TABLE(
    	luokka varchar(20),
        kokonaismyyntihinta decimal(6,2),
        keskihinta decimal(6,2)   
    )
    
    Esimerkkikutsu
    > SELECT * FROM hae_myytavien_hintatiedot();
